What is a Function?

A function is a block of organized, reusable code used to perform a single, related action. They are the fundamental building blocks of Python programs, helping you create modular, readable, and efficient code.

Why use functions?

  • Modularity: Break down complex problems into smaller, manageable sub-problems.
  • Reusability: Write code once and use it multiple times (DRY - Don't Repeat Yourself).
  • Readability: Make your code easier to understand and maintain.
  • Debugging: Isolate issues to specific functions, simplifying the debugging process.

Defining and Calling a Function

You define a function with the def keyword, followed by a name, parentheses, and a colon. To use the function, you "call" it by its name. Click 'Run' to see how the code below works.

def greet(name):
    """This function greets the person passed in."""
    print(f"Hello, {name}!")

# Now, let's call the function
greet("Alice")

Function Arguments

Arguments are values passed into a function when it's called. Python supports several types of arguments, giving you flexibility in how you design your functions. Explore the different types below.

1. Positional Arguments

These are the most common type. The arguments are passed in order, and the number of arguments must match the function's parameters.

def describe_person(name, age):
    print(f"Name: {name}, Age: {age}")

describe_person("Bob", 30)

2. Default Arguments

You can provide a default value for a parameter. If the argument isn't passed during the function call, the default value is used.

def display_product(item, price=100):
    print(f"Item: {item}, Price: ${price}")

display_product("Laptop")
display_product("Mouse", 250)

3. Keyword Arguments

You can specify which parameter an argument corresponds to by using its name. This allows you to pass arguments out of order.

def student_details(name, roll_no, grade):
    print(f"Student: {name}, Roll No: {roll_no}, Grade: {grade}")

student_details(grade="A", name="Divya", roll_no=101)

4. Variable-Length Arguments (*args)

Use `*args` to pass a variable number of positional arguments to a function. They are received as a tuple.

def calculate_average(*numbers):
    avg = sum(numbers) / len(numbers) if numbers else 0
    print(f"The average is: {avg}")

calculate_average(5, 10, 15, 20)

Return Statement

The return statement is used to exit a function and send a value back to where it was called. If no return statement is given, a function returns None.

def add_numbers(a, b):
    return a + b

result = add_numbers(10, 20)
print(f"The returned value is: {result}")

Scope of Variables

Scope refers to the region of the program where a variable can be accessed. Understanding scope is crucial for avoiding bugs.

Visualizing Scope

Global Scope

global_var = "I am global"

Local Scope (inside my_function)

local_var = "I am local"

Can access both `local_var` and `global_var` here.

Only `global_var` can be accessed here. `local_var` is not defined in this scope.

The `global` Keyword

To modify a global variable from inside a function, you must use the global keyword.

count = 0 # Global variable

def increment_count():
    global count
    count += 1
    print(f"Inside function: {count}")

print(f"Before: {count}")
increment_count()
print(f"After: {count}")

Advanced Concepts

Let's look at different types of functions and a powerful concept called recursion.

Types of Functions

  • Built-in Functions: Pre-defined in Python and always available (e.g., `print()`, `len()`, `type()`).
  • User-defined Functions: Functions you create using the `def` keyword.
  • Module Functions: Functions from modules that you must `import` (e.g., `math.sqrt()`).

Recursion

A recursive function is one that calls itself. It must have a **base case** to stop the recursion and prevent an infinite loop. Here's a classic example: calculating a factorial.

def factorial(n):
    # Base case:
    if n == 0 or n == 1:
        return 1
    # Recursive step:
    else:
        return n * factorial(n - 1)

result = factorial(5)
print(f"The factorial of 5 is: {result}")
# The calculation goes like: 5 * 4 * 3 * 2 * 1
